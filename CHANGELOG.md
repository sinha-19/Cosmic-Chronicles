# Changelog

All notable changes to Cosmic Chronicles will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-XX-XX

### Added
- Initial release of Cosmic Chronicles
- Procedural universe generation system with seed support
- Turn-based combat system with attack, heal, and shield mechanics
- Character progression and training system
- Navigation system (villages, castles, planets)
- NPC interaction system with procedurally-generated characters
- Save/Load game functionality with multiple save slots
- Random adventure generation
- Time manipulation through Zarathorn's Time Machine
- Boss summoning system
- God and Vorathar as ultimate challenges
- Universe switching capability
- Universe Book with lore and story
- Special locations (Wizard, President's Hall, Unknown Place)
- Easter eggs and hidden features
- Auto-save functionality
- Color-coded terminal interface
- Cross-platform support (Windows, Linux, macOS)

### Features
- Infinite procedurally-generated universes
- Unique character name generation
- Dynamic power scaling system
- Multiple game endings
- Hidden dialogue options
- Year progression system
- Character aging mechanics

## How to Update

When making changes to the game:

1. Update this CHANGELOG.md file
2. Increment the version number in game.py (add a version constant)
3. Tag the release in git: `git tag -a v1.0.0 -m "Version 1.0.0"`
4. Push tags: `git push origin --tags`
5. Create a GitHub release with the changelog notes
